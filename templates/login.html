Pepwave Cellular Usage Tool

This tool is used to track the cellular usage for C2R's Pepwave devices.

STEP 1

Select the dates you want to see the total cellular usage for.

STEP 2

Click search

THINGS TO NOTE:

  • You will only see the cellular usage for devices that are over 3GB and above.
  • You cannot see data past 60 days from the current date.
  • You can correlate the dates to a Job's Event Rental Start and End date if you want to ensure there aren't any Pepwaves going over their usage.
